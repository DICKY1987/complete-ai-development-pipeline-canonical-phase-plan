# Software Development Profile (Python Variant)

This profile configures the Universal Execution Templates Framework for Python software development projects.

## When to Use This Profile

Use `software-dev-python` when your project:
- Is primarily Python code (>60% .py files)
- Has tests (pytest, unittest, etc.)
- Uses version control (git)
- Follows software development best practices
