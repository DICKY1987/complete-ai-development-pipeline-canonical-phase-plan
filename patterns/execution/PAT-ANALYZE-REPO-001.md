# PAT-ANALYZE-REPO-001: Analyze Repository Context

## Purpose
Quickly understand the repository layout and current state before making changes.

## Steps
1. List key directories and files relevant to the task.
2. Read project-level instructions or contracts.
3. Identify existing templates and scripts to reuse.
4. Note current git status to avoid clobbering user changes.

## Success Criteria
- Clear inventory of relevant files.
- No modifications made during analysis.
- Scope and constraints identified.
