## After Installation

### Reload PowerShell Profile

```powershell

```

### Available Commands

```powershell
aider-wsl <RepoName>      # Launch Aider for a specific repo
aider-here-wsl            # Launch Aider in current directory
codex-wsl <RepoName>      # Launch Codex for a specific repo
codex-here-wsl            # Launch Codex in current directory
wsl-code [RepoName]       # Open WSL terminal in repo
wsl-repos                 # List repos in ~/code
```

### Clone a Repository

```bash
# Open WSL
wsl -d Ubuntu

# Clone into ~/code
cd ~/code
git clone https://github.com/your-org/your-repo.git
```

### Launch Aider

```powershell
# From Windows PowerShell
aider-wsl your-repo
```

## Troubleshooting

### WSL won't install
- Ensure Windows is fully updated
- Enable virtualization in BIOS
- Run PowerShell as Administrator

### "Ubuntu not found"
- Open Microsoft Store
- Search "Ubuntu"
- Install/Launch Ubuntu manually

### Aider shows broken characters
- Use Windows Terminal (not cmd.exe)
- Set font to "Cascadia Code" or "Cascadia Mono"
- Verify locale in WSL: `locale` (should show UTF-8)

### PowerShell functions don't work
```powershell
# Reload profile
. $PROFILE

# Or restart PowerShell
```

### Need to start over?
```powershell
# Remove Ubuntu and start fresh
wsl --unregister Ubuntu

# Then run Step 1 again
.\WSL-Install-Step1-ADMIN-REQUIRED.ps1
```
