## Troubleshooting

### "Aider command not found"
```bash
wsl -d Ubuntu
source ~/.venvs/aider/bin/activate
aider --version
```

### "Cannot connect to Ubuntu"
```powershell
wsl --list --verbose
# Should show "Ubuntu" as "Running"

# If not running:
wsl -d Ubuntu
```

### "API key not working"
```bash
wsl -d Ubuntu
echo $OPENAI_API_KEY
# Should show your key (or empty if not set)

# If empty:
source ~/.bashrc
echo $OPENAI_API_KEY
```

### Functions don't work
```powershell
# Reload profile
. $PROFILE

# Or check if they exist
Get-Command aider-wsl
```
