# File Watcher

This directory contains file watching utilities for monitoring changes to project files.

## Contents

### `Watch-PhaseDocs.ps1`
PowerShell script that monitors phase documentation files for changes and triggers appropriate actions.

## Purpose

The file watcher tools provide real-time monitoring capabilities for:
- Detecting changes to phase documentation
- Triggering automated workflows when files are modified
- Maintaining synchronization between related files
- Enabling reactive development workflows

## Usage

Run `Watch-PhaseDocs.ps1` to start monitoring phase documentation files. The watcher will continuously monitor for changes and trigger configured actions when modifications are detected.

This is particularly useful during development when you want automatic updates or validations to occur whenever documentation is changed.
