# Visual Diagrams

This directory contains all visual diagrams, flowcharts, and graphical documentation for the AI development pipeline system.

## Contents

### Diagram Files
- `AUTOMATION_CHAIN_DIAGRAM.txt` - Text representation of automation chain flow
- `Configuration Cascade.png` - Visual representation of configuration inheritance
- `Error Detection & Recovery Flow.png` - Error handling flowchart
- `Folder Interaction Heatmap.png` - Visualization of folder interactions
- `Information Flow by Phase.png` - Phase-based information flow diagram
- `Module Dependency Graph.png` - Module dependency visualization
- `Task Lifecycle State Machine.png` - State machine for task lifecycle
- `Tool Adapter Pattern.png` - Tool adapter pattern architecture
- `Workstream Execution Timeline.png` - Timeline of workstream execution
- `E2E_Process_Flow.drawio` - End-to-end process flow (editable)
- `new 3.xml` - Additional diagram data

### Mermaid Diagrams (`.mmd` files)
- `data-flow-aim-integration.mmd` - AIM integration data flow
- `data-flow-database.mmd` - Database data flow
- `data-flow-error-detection.mmd` - Error detection data flow
- `data-flow-workstream.mmd` - Workstream data flow
- `directory-structure.mmd` - Directory structure diagram
- `integration-overview.mmd` - System integration overview
- `module-dependencies.mmd` - Module dependencies diagram

### Documentation
- `DIAGRAM_CREATION_GUIDE.md` - Guide for creating new diagrams
- `DOC_ERROR_ESCALATION_DIAGRAM.md` - Error escalation documentation
- `DOC_SPEC_INTEGRATION_DIAGRAM.md` - Specification integration documentation
- `DOC_TASK_LIFECYCLE_DIAGRAM.md` - Task lifecycle documentation
- `DOC_TOOL_SELECTION_DIAGRAM.md` - Tool selection process documentation
- `FOLDER_LIFECYCLE_FLOW.md` - Folder lifecycle flow documentation
- `SYSTEM_VISUAL_DIAGRAMS.md` - Overall system diagram documentation
- `TASK_LIFECYCLE.md` - Task lifecycle details
- `UI_FLOW_DIAGRAM.md` - UI flow documentation
- `PAT-DIAGRAM-DRAWIO-RENDER-001.md` - Pattern diagram rendering documentation

## Purpose

This directory serves as the central repository for all visual documentation, making it easy to:
- Understand system architecture visually
- View process flows and state machines
- Analyze dependencies and relationships
- Learn system behavior through diagrams

## Diagram Formats

- **PNG/Image files**: Rendered diagrams ready for viewing
- **Mermaid (.mmd)**: Text-based diagrams that can be rendered with Mermaid
- **DrawIO (.drawio)**: Editable diagrams that can be opened with draw.io
- **XML**: Diagram data in XML format
- **TXT**: Text-based diagram representations

## Usage

Refer to `DIAGRAM_CREATION_GUIDE.md` for instructions on creating and maintaining diagrams in this directory.
